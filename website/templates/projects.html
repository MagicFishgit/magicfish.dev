prettyweather

A weather website that displays weather information for any location using openweathermaps API, GCP Maps API, Python3, Flask and Jinja2
This was quite fun to make. Had lots of practice with API calls, parsing and manipulating data from JSON. My knowledge of object orientation came in very handy here. The website will show WEATHER and POLLUTION data from any city/place you search. I used Googles Map API to put a map as the background which changes to the location that you are searching.
Technologies used:
  • Python3
  • Jinja2
  • Flask
  • Bootstrap 5
  • HTML
  • CSS
  • SCSS
  • JavaScript
  • OpenWeathermaps API
  • Google Maps API
