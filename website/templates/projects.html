A flappy bird clone using machine learning (NEAT-Python) to play the game optimally and Pygame for the visuals.
The main objective here is to practice coding in Python, learn about machine learning and getting used to Python object orientation. This was fun to create and had me deal with a quite a few interesting bugs as well as help ingrain Python syntax and OOP principals.
